# My personal dotfiles

Install with Stow with the commnad
`stow --target=${HOME} */` 

## Specifications

- WM: i3-gaps
- Bar: polybar
- Compositor: picom
- Terminal: Alacritty
- Notifications: Dunst
- App launcher: Rofi
- Shell: ZSH
